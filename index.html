Fase 01
Herramientas y preparación
Todo lo que debes instalar y descargar antes de empezar
1.1 Instala las herramientas necesarias en tu equipo de trabajo ▼
Estas tres herramientas son obligatorias. Sin ellas no puedes extraer la ISO, modificar el WIM ni crear la ISO final.

7-Zip

Para extraer el contenido de la ISO. Descarga desde 7-zip.org → instala el .exe de 64 bits.

Windows ADK — solo "Deployment Tools"

Incluye DISM (para montar y modificar el WIM) y oscdimg (para crear la ISO final). Busca en Google "Windows ADK download Microsoft" → descarga adksetup.exe → al instalar marca únicamente Deployment Tools. Pesa ~200 MB.

Notepad++

Imprescindible para editar el autounattend.xml sin corromper la codificación UTF-8. Descarga desde notepad-plus-plus.org.

Rufus

Para grabar la ISO en USB. Descarga desde rufus.ie. Es un solo .exe, sin instalación.

  • 7-Zip instalado
  • Windows ADK instalado (solo "Deployment Tools")
  • Notepad++ instalado
  • Rufus descargado
1.2 Descarga la ISO oficial de Windows 11 y los instaladores offline ▼
Crítico: Chrome y Java deben ser sus versiones offline. Los instaladores online no funcionan sin internet y no instalarán nada.

ISO de Windows 11

Ve a microsoft.com/es-es/software-download/windows11 → "Descargar imagen de disco de Windows 11 (ISO para dispositivos x64)" → Español → descarga. Pesa ~5.8 GB.

Google Chrome — instalador offline MSI (obligatorio)

El ChromeSetup.exe que descarga Google normalmente es un instalador online que requiere internet. Para uso sin internet necesitas el MSI enterprise:

Ve a chromeenterprise.google/intl/es_es/browser/download/ → Windows 64-bit → MSI. El archivo se llamará googlechromestandaloneenterprise64.msi.

Java — instalador offline (obligatorio)

El JavaSetup8u301.exe estándar también es online. Necesitas el instalador offline:

Ve a java.com/es/download/manual.jsp → Windows Offline (64 bits). El archivo se llamará jre-8uXXX-windows-x64.exe.

Fase 02
Extraer la ISO base
Descomprimir el contenido para poder modificarlo
2.1 Extrae la ISO de Windows 11 con 7-Zip ▼
¿Por qué extraer? Una ISO es un archivo comprimido. Para agregar archivos y modificar el WIM hay que extraer todo a una carpeta normal primero.

Clic derecho sobre el archivo .iso descargado → 7-Zip → Extraer en... → destino: C:\ProyectoISO\ISO_fuente\

Tarda 3-7 minutos. Al terminar verás estas carpetas dentro de ISO_fuente:

C:\ProyectoISO\ISO_fuente\ ├── boot\ ├── efi\ ├── sources\ │ └── install.wim ← el sistema operativo └── support\
ℹ️
El archivo más importante es sources\install.wim. Aquí se inyectarán los instaladores en la Fase 4.
2.2 Verifica los índices del WIM ▼
Crítico: Una ISO de Windows 11 contiene varias ediciones (Home, Pro, Education, etc.). Cada una tiene un número de índice. Debes saber cuál es la edición que quieres instalar para modificar el índice correcto.
CMD — Administrador
DISM /Get-ImageInfo /ImageFile:"C:\ProyectoISO\ISO_fuente\sources\install.wim"

Verás una lista como esta. Anota el índice de la edición que quieres:

ÍndiceEdiciónUsar
1Windows 11 HomeSi quieres Home
4Windows 11 Pro✓ El que usamos
6Windows 11 Pro for WorkstationsNo confundir con Pro
🚫
Error que cometimos
Al principio usamos el índice 6 creyendo que era Pro, pero era "Pro for Workstations". El índice correcto para Windows 11 Pro es el 4. Verifica siempre el nombre antes de continuar.

Fase 03
Preparar el script de instalación
El archivo instalar.cmd con los parámetros silenciosos correctos
3.1 Crea el archivo instalar.cmd con parámetros verificados ▼
Parámetros silenciosos verificados: Cada instalador usa un motor diferente con sus propios comandos. Estos fueron probados y confirmados uno por uno.
ProgramaMotorParámetro silencioso
7-ZipNSIS/S (mayúscula obligatoria)
MoffFree CalcInno Setup/VERYSILENT /SUPPRESSMSGBOXES /NORESTART
K-LiteInno Setup/VERYSILENT /SUPPRESSMSGBOXES /NORESTART
Foxit ReaderInno Setup/VERYSILENT /SUPPRESSMSGBOXES /NORESTART
Java offlineOracle/s REBOOT=0 SPONSORS=0
Chrome MSIMSI (msiexec)/qn /norestart ALLUSERS=1
FirefoxMozilla-ms

Abre Notepad++, crea un archivo nuevo y pega exactamente este contenido:

instalar.cmd
@echo off
:: Elimina RunOnce para que no vuelva a ejecutar
reg delete "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\RunOnce" /v "InstalarProgramas" /f >nul 2>&1

:: Espera 30 segundos a que el escritorio cargue completamente
timeout /t 30 /nobreak >nul

echo [%DATE% %TIME%] ===== INICIO ===== >> C:\install_log.txt
cd /d "C:\Windows\Setup\Scripts\Installers"

:: 7-Zip — Motor NSIS, /S en MAYUSCULA
start /wait /b "" "7z1604-x64.exe" /S
echo [%DATE% %TIME%] 7-Zip: %ERRORLEVEL% >> C:\install_log.txt

:: MoffFree Calculator — Motor Inno Setup
start /wait /b "" "MoffFreeCalcSetup.exe" /VERYSILENT /SUPPRESSMSGBOXES /NORESTART
echo [%DATE% %TIME%] MoffFree: %ERRORLEVEL% >> C:\install_log.txt

:: K-Lite Codec Pack — Motor Inno Setup
start /wait /b "" "K-Lite_Codec_Pack_1196_Mega.exe" /VERYSILENT /SUPPRESSMSGBOXES /NORESTART
echo [%DATE% %TIME%] K-Lite: %ERRORLEVEL% >> C:\install_log.txt

:: Foxit Reader — Motor Inno Setup
start /wait /b "" "FoxitReader1011_L10N_Setup_Prom.exe" /VERYSILENT /SUPPRESSMSGBOXES /NORESTART
echo [%DATE% %TIME%] Foxit: %ERRORLEVEL% >> C:\install_log.txt

:: Java offline — reemplaza XXXX por tu version (ej: 8u491)
start /wait /b "" "jre-8u491-windows-x64.exe" /s REBOOT=0 SPONSORS=0
echo [%DATE% %TIME%] Java: %ERRORLEVEL% >> C:\install_log.txt

:: Google Chrome MSI offline — msiexec para archivos .msi
start /wait /b "" msiexec /i "googlechromestandaloneenterprise64.msi" /qn /norestart ALLUSERS=1
echo [%DATE% %TIME%] Chrome: %ERRORLEVEL% >> C:\install_log.txt

:: Firefox — ruta completa obligatoria por el espacio en el nombre
start /wait /b "" "C:\Windows\Setup\Scripts\Installers\Firefox Setup 85.0.2.exe" -ms
echo [%DATE% %TIME%] Firefox: %ERRORLEVEL% >> C:\install_log.txt

echo [%DATE% %TIME%] ===== COMPLETADO ===== >> C:\install_log.txt
exit /b 0

Guarda el archivo en Notepad++:

→ Archivo → Guardar como → tipo: Todos los archivos (*.*) → nombre: instalar.cmd → guárdalo en: C:\ProyectoISO\

🚫
Verificar codificación en Notepad++
Menú Codificación → UTF-8 (sin BOM). Si dice otra cosa, cámbiala antes de guardar. La codificación incorrecta hace que Windows ignore el archivo.

Fase 04
Inyectar instaladores en el WIM
El método que garantiza que los archivos siempre lleguen al disco
4.1 ¿Por qué inyectar en el WIM y no usar $OEM$? ▼
Lección aprendida en este proceso: La carpeta $OEM$ depende de cómo Rufus graba el USB. En varias pruebas, Rufus no copió el $OEM$ correctamente al disco durante la instalación. Inyectar los archivos directamente dentro del install.wim garantiza que siempre lleguen a C:\ porque van dentro del propio sistema operativo.
MétodoConfiabilidadResultado en pruebas
$OEM$ en ISO❌ InestableRufus no copiaba los archivos al disco
FirstLogonCommands❌ InestableWindows 11 22H2+ no lo dispara confiablemente
SetupComplete.cmd❌ InestableNo se ejecuta automáticamente en Win11 reciente
WIM + RunOnce✓ FuncionaInstaladores siempre llegan y se ejecutan
4.2 Limpia montajes anteriores y monta el WIM ▼

Primero limpia cualquier montaje anterior que haya quedado colgado:

CMD — Administrador
DISM /Cleanup-Mountpoints

Luego monta el WIM. Usa el índice que identificaste (para Windows 11 Pro es el 4):

CMD — Administrador
DISM /Mount-Image /ImageFile:"C:\ProyectoISO\ISO_fuente\sources\install.wim" /Index:4 /MountDir:"C:\ProyectoISO\WIM_mount"

Tarda entre 5 y 15 minutos. Cuando termine verás: The operation completed successfully.

⚠️
Si sale error 0xc1420127 significa que el WIM ya está montado de una sesión anterior. Ejecuta DISM /Cleanup-Mountpoints y vuelve a intentarlo.
4.3 Copia los instaladores y el script dentro del WIM ▼
CMD — Administrador
:: Crea la carpeta de instaladores dentro del WIM
mkdir "C:\ProyectoISO\WIM_mount\Windows\Setup\Scripts\Installers"

:: Copia los 7 instaladores — ajusta la ruta a donde los tengas
xcopy /E /H /Y "C:\Users\wbolivar\Desktop\Windows 11 desatendido\Instaladores\*.*" "C:\ProyectoISO\WIM_mount\Windows\Setup\Scripts\Installers\"

:: Copia el instalador offline de Chrome
copy /Y "C:\Users\wbolivar\Downloads\googlechromestandaloneenterprise64.msi" "C:\ProyectoISO\WIM_mount\Windows\Setup\Scripts\Installers\"

:: Copia el instalador offline de Java
copy /Y "C:\Users\wbolivar\Downloads\jre-8u491-windows-x64.exe" "C:\ProyectoISO\WIM_mount\Windows\Setup\Scripts\Installers\"

:: Copia el script de instalacion
copy /Y "C:\ProyectoISO\instalar.cmd" "C:\ProyectoISO\WIM_mount\Windows\Setup\Scripts\instalar.cmd"
ℹ️
Ajusta las rutas de origen según donde tengas guardados los archivos en tu equipo.
4.4 Registra RunOnce en el registro del WIM ▼
¿Qué es RunOnce? Es una clave del registro de Windows que guarda comandos para ejecutar una sola vez al próximo inicio de sesión. Al registrarla dentro del WIM montado, queda grabada en el sistema operativo que se instalará. Al primer login del usuario GTA, Windows ejecuta automáticamente el instalar.cmd y luego borra la entrada.
CMD — Administrador
:: Carga el registro del WIM montado en HKLM temporal
reg load HKLM\WIM_SOFTWARE "C:\ProyectoISO\WIM_mount\Windows\System32\config\SOFTWARE"

:: Registra el comando que instalara los programas al primer login
reg add "HKLM\WIM_SOFTWARE\Microsoft\Windows\CurrentVersion\RunOnce" /v "InstalarProgramas" /t REG_SZ /d "cmd /c C:\Windows\Setup\Scripts\instalar.cmd" /f

:: Descarga el registro del WIM
reg unload HKLM\WIM_SOFTWARE
✅
Los tres comandos deben responder The operation completed successfully.. Si el tercero falla, espera unos segundos y vuelve a ejecutar reg unload HKLM\WIM_SOFTWARE.
4.5 Desmonta y guarda el WIM ▼
CMD — Administrador
DISM /Unmount-Image /MountDir:"C:\ProyectoISO\WIM_mount" /Commit

Tarda entre 5 y 20 minutos. Espera a que aparezca The operation completed successfully. antes de continuar.

🚫
No cierres el CMD ni apagues el equipo mientras desmonta. Si se interrumpe el WIM puede corromperse y tendrás que volver a extraer la ISO desde cero.

Fase 05
Crear el autounattend.xml
El archivo que automatiza toda la instalación de Windows
5.1 Crea el autounattend.xml en Notepad++ ▼
¿Qué hace este archivo? Responde automáticamente todas las preguntas del instalador de Windows: idioma, particionar el disco, nombre del equipo, crear el usuario GTA y saltar toda la pantalla de bienvenida. Windows lo lee si lo encuentra en la raíz del USB al arrancar.
🚫
Tres reglas críticas aprendidas en este proceso
1. La primera línea DEBE ser <?xml version="1.0" encoding="utf-8"?> — sin ella Windows ignora el archivo.
2. La codificación en Notepad++ DEBE ser UTF-8 (sin BOM).
3. Debe estar en la raíz del USB, no dentro de ninguna subcarpeta.

Abre Notepad++ → menú Codificación → UTF-8 (sin BOM) → pega este contenido:

autounattend.xml
<?xml version="1.0" encoding="utf-8"?>
<unattend xmlns="urn:schemas-microsoft-com:unattend">

  <!-- windowsPE: corre ANTES de instalar. Define idioma, disco e imagen. -->
  <settings pass="windowsPE">
    <component name="Microsoft-Windows-International-Core-WinPE"
               processorArchitecture="amd64"
               publicKeyToken="31bf3856ad364e35"
               language="neutral" versionScope="nonSxS"
               xmlns:wcm="http://schemas.microsoft.com/WMIConfig/2002/State">
      <SetupUILanguage><UILanguage>es-CO</UILanguage></SetupUILanguage>
      <InputLocale>es-CO</InputLocale>
      <SystemLocale>es-CO</SystemLocale>
      <UILanguage>es-CO</UILanguage>
      <UserLocale>es-CO</UserLocale>
    </component>
    <component name="Microsoft-Windows-Setup"
               processorArchitecture="amd64"
               publicKeyToken="31bf3856ad364e35"
               language="neutral" versionScope="nonSxS"
               xmlns:wcm="http://schemas.microsoft.com/WMIConfig/2002/State">
      <UserData><AcceptEula>true</AcceptEula></UserData>
      <DiskConfiguration>
        <Disk wcm:action="add">
          <DiskID>0</DiskID>
          <WillWipeDisk>true</WillWipeDisk>
          <CreatePartitions>
            <CreatePartition wcm:action="add"><Order>1</Order><Type>EFI</Type><Size>300</Size></CreatePartition>
            <CreatePartition wcm:action="add"><Order>2</Order><Type>MSR</Type><Size>16</Size></CreatePartition>
            <CreatePartition wcm:action="add"><Order>3</Order><Type>Primary</Type><Extend>true</Extend></CreatePartition>
          </CreatePartitions>
          <ModifyPartitions>
            <ModifyPartition wcm:action="add"><Order>1</Order><PartitionID>1</PartitionID><Format>FAT32</Format><Label>System</Label></ModifyPartition>
            <ModifyPartition wcm:action="add"><Order>2</Order><PartitionID>2</PartitionID></ModifyPartition>
            <ModifyPartition wcm:action="add"><Order>3</Order><PartitionID>3</PartitionID><Format>NTFS</Format><Label>Windows</Label><Letter>C</Letter></ModifyPartition>
          </ModifyPartitions>
        </Disk>
      </DiskConfiguration>
      <ImageInstall>
        <OSImage>
          <InstallTo><DiskID>0</DiskID><PartitionID>3</PartitionID></InstallTo>
          <WillShowUI>Never</WillShowUI>
        </OSImage>
      </ImageInstall>
    </component>
  </settings>

  <!-- specialize: nombre del equipo y zona horaria -->
  <settings pass="specialize">
    <component name="Microsoft-Windows-Shell-Setup"
               processorArchitecture="amd64"
               publicKeyToken="31bf3856ad364e35"
               language="neutral" versionScope="nonSxS"
               xmlns:wcm="http://schemas.microsoft.com/WMIConfig/2002/State">
      <ComputerName>PC-GTA</ComputerName>
      <TimeZone>SA Pacific Standard Time</TimeZone>
    </component>
  </settings>

  <!-- oobeSystem: usuario GTA, salta bienvenida, auto-login -->
  <settings pass="oobeSystem">
    <component name="Microsoft-Windows-Shell-Setup"
               processorArchitecture="amd64"
               publicKeyToken="31bf3856ad364e35"
               language="neutral" versionScope="nonSxS"
               xmlns:wcm="http://schemas.microsoft.com/WMIConfig/2002/State">
      <OOBE>
        <HideEULAPage>true</HideEULAPage>
        <HideLocalAccountScreen>false</HideLocalAccountScreen>
        <HideOnlineAccountScreens>true</HideOnlineAccountScreens>
        <HideWirelessSetupInOOBE>true</HideWirelessSetupInOOBE>
        <NetworkLocation>Work</NetworkLocation>
        <ProtectYourPC>3</ProtectYourPC>
        <SkipMachineOOBE>true</SkipMachineOOBE>
        <SkipUserOOBE>true</SkipUserOOBE>
      </OOBE>
      <UserAccounts>
        <LocalAccounts>
          <LocalAccount wcm:action="add">
            <Name>GTA</Name>
            <Group>Administrators</Group>
            <DisplayName>GTA</DisplayName>
            <Password><Value></Value><PlainText>true</PlainText></Password>
          </LocalAccount>
        </LocalAccounts>
      </UserAccounts>
      <AutoLogon>
        <Username>GTA</Username>
        <LogonCount>1</LogonCount>
        <Enabled>true</Enabled>
      </AutoLogon>
    </component>
  </settings>

</unattend>

Guarda como: Archivo → Guardar como → tipo: Todos los archivos → nombre: autounattend.xml → guarda en C:\ProyectoISO\

Fase 06
Generar la ISO final
Empaquetar todo en un archivo .iso booteable
6.1 Genera la ISO con oscdimg ▼
¿Por qué oscdimg y no comprimir la carpeta? Una ISO booteable necesita sectores de arranque especiales (uno para BIOS, otro para UEFI) que oscdimg añade automáticamente. Sin ellos el USB no arranca.
CMD — Administrador
"C:\Program Files (x86)\Windows Kits\10\Assessment and Deployment Kit\Deployment Tools\amd64\Oscdimg\oscdimg.exe" -m -o -u2 -udfver102 -bootdata:2#p0,e,b"C:\ProyectoISO\ISO_fuente\boot\etfsboot.com"#pEF,e,b"C:\ProyectoISO\ISO_fuente\efi\microsoft\boot\efisys.bin" "C:\ProyectoISO\ISO_fuente" "C:\ProyectoISO\Windows11_GTA_FINAL.iso"

Tarda 5-15 minutos. Cuando termine verás Done. y se habrá creado: C:\ProyectoISO\Windows11_GTA_FINAL.iso

7.1 Graba la ISO con Rufus ▼

Abre Rufus y configura exactamente así:

CampoValor

⚠️
Si aparece una ventana de "Personalizar la instalación de Windows", deja todo sin marcar y clic en OK. Cualquier opción marcada puede interferir con el proceso.

7.2 Copia el autounattend.xml al USB después de grabar ▼
¿Por qué copiarlo después y no antes? Rufus puede sobrescribir o modificar archivos en la raíz del USB al grabar. Copiarlo al final garantiza que el archivo quedará exactamente como lo creaste.

Reemplaza W: con la letra que tenga tu USB:

CMD — Administrador
copy /Y "C:\ProyectoISO\autounattend.xml" "W:\autounattend.xml"

Verifica que el archivo quedó en la raíz del USB (no dentro de ninguna subcarpeta).
